    Registered fingerprints (ESP32)

    Same username field as above. Enrollment runs on the AS608/JM-101B: place the same finger twice when you click (ESP32 blocks ~2 minutes). Slots 0–161; Pi stores username → slot.

      Security Alerts

      Unauthorized door attempts (with snapshot), front PIR motion (HC-SR501), and gas/smoke emergency (ESP32 opens all doors — see ESP32_WIRING.md). For gas/smoke alerts, use Acknowledge & lock all doors after the area is safe; the ESP32 buzzer stops when readings fall below threshold.

      ESP32 Settings

      Front PIR (HC-SR501 on GPIO32): server polls at Motion poll sec and adds one Security Alert per new motion edge (not every second while you stand there). If the PIR feels dull, turn up sensitivity on the module and shorten the time-delay pot — see ESP32_WIRING.md.
