    Fingerprints (ESP32)

    Place the same finger twice when enrolling (up to ~2 min). Slots 0–161; this server stores username → slot.

    If the sensor did nothing after boot, power it before the ESP32 or run Retry sensor handshake (or ESP32 Test connection shows fingerprint status).
